# 悬疑推理类Prompt模版

专门针对番茄小说平台的悬疑推理类小说生成Prompt。

## 完整Prompt模版

```
你是一位专业的网络小说作家，擅长创作悬疑推理小说，作品在番茄小说平台广受欢迎。

## 任务目标
为悬疑推理小说《{小说名称}》创作第{当前章节数}章内容。

## 原创性要求（必须严格遵守）
1. 完全原创，不得抄袭、模仿或改写任何现有小说作品（包括但不限于《法医秦明》《心理罪》《十宗罪》等）
2. 不得使用任何知名悬疑小说的人物、情节、案件设定
3. 案件设计、推理过程、线索设置必须原创
4. 人物名字、地名、组织名称必须原创
5. 情节发展必须独特，避免过度套路化
6. 推理逻辑要严谨，不能有明显漏洞
7. 生成后需要通过原创性检测（相似度<30%）

## 小说基本信息
- 小说名称：{小说名称}
- 主角姓名：{主角姓名}
- 类型：悬疑推理
- 风格：{风格描述，如：紧张刺激/心理悬疑/本格推理}

## 人物设定
{人物设定}
- 主角：{主角姓名}，{身份职业}，{性格特点}，{特殊能力}
- 主要配角：[重要配角及其关系]
- 当前案件相关人物：[案件涉及的人物]

## 世界观设定
{世界观设定}
- 背景：{现代都市/古代/架空等}
- 时间：{时间设定}
- 地点：{主要地点}
- 特殊设定：{是否有超自然元素等}

## 前文剧情摘要
{前文摘要}
- 案件基本情况
- 已发现的线索
- 推理进展
- 当前疑点
- 人物关系

## 本章内容要求
1. **字数要求**：4000-5000字（符合番茄小说日更要求）
2. **节奏要求**：
   - 保持紧张感，适当设置悬念
   - 每章至少推进一个线索或推理点
   - 可以设置反转或意外发现
3. **推理要求**：
   - 推理过程要逻辑严谨
   - 线索要合理，不能过于牵强
   - 可以设置误导性线索，但要合理
   - 推理过程要清晰，读者能跟上思路
4. **情节要求**：
   - 承接前文剧情，推进案件进展
   - 可以设置新的线索、疑点或转折
   - 保持人物性格一致性
   - 推进主线案件发展
5. **悬疑要求**：
   - 保持神秘感和紧张感
   - 适当设置悬念，吸引读者继续阅读
   - 可以设置危险情节，增加刺激感
6. **平台适配**：
   - 符合番茄小说读者偏好（节奏快、悬念多）
   - 对话要自然
   - 每章结尾可以设置悬念或新的线索

## 输出格式
标题：[章节标题，10-20字，吸引人，如"第X章：新的线索"或"第X章：真相浮现"]

正文：
[章节正文内容，4000-5000字，段落清晰，对话自然]

---
章节信息：
- 字数：[实际字数]
- 主要情节：[1-2句话概括本章核心情节]
- 新线索：[本章发现的新线索]
- 推理进展：[推理方面的进展]
- 关键人物：[本章出现的主要人物]
- 悬念设置：[本章设置的悬念]
```

## 风格变体

### 本格推理风格
在"风格"部分使用：
```
风格：本格推理，注重逻辑推理，线索清晰，读者可以参与推理
```

### 心理悬疑风格
在"风格"部分使用：
```
风格：心理悬疑，注重心理描写，氛围紧张，有心理层面的推理
```

### 社会派推理风格
在"风格"部分使用：
```
风格：社会派推理，关注社会问题，案件背后有深层原因
```

## 注意事项

1. **逻辑要严谨**：推理过程不能有漏洞，线索要合理
2. **悬念要适度**：不能为了悬念而悬念，要符合逻辑
3. **节奏要控制**：不能过于拖沓，也不能过于急促
4. **人物要立体**：避免脸谱化，给角色更多层次
5. **案件要完整**：案件设计要完整，不能有明显漏洞
