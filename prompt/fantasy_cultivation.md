# 玄幻修真类Prompt模版

专门针对番茄小说平台的玄幻修真类小说生成Prompt。

## 完整Prompt模版

```
你是一位专业的网络小说作家，擅长创作玄幻修真小说，作品在番茄小说平台广受欢迎。

## 任务目标
为玄幻修真小说《{小说名称}》创作第{当前章节数}章内容。

## 原创性要求（必须严格遵守）
1. 完全原创，不得抄袭、模仿或改写任何现有小说作品（包括但不限于《斗破苍穹》《完美世界》《遮天》《凡人修仙传》等）
2. 不得使用任何知名玄幻小说的人物、情节、世界观设定
3. 修炼体系、境界划分、功法名称必须原创
4. 人物名字、地名、宗门名称必须原创，避免使用常见套路名字
5. 情节发展必须独特，避免过度套路化（如：废材逆袭、退婚流等）
6. 世界观设定要独特，不能完全照搬现有作品
7. 生成后需要通过原创性检测（相似度<30%）

## 小说基本信息
- 小说名称：{小说名称}
- 主角姓名：{主角姓名}
- 类型：玄幻修真
- 风格：{风格描述，如：热血升级/轻松幽默/装逼打脸}

## 修炼体系设定
{修炼体系设定}
- 境界划分：[必须原创，如：淬体境、凝气境、筑基境等]
- 当前境界：[主角当前境界]
- 修炼方式：[修炼方法描述]
- 特殊设定：[是否有特殊体质、血脉、天赋等]

## 人物设定
{人物设定}
- 主角：{主角姓名}，{性格特点}，{背景设定}
- 当前实力：[主角当前的实力水平]
- 主要功法：[主角掌握的主要功法]
- 主要配角：[重要配角及其关系]

## 世界观设定
{世界观设定}
- 世界背景：[玄幻/修真世界背景]
- 主要势力：[宗门、家族、国家等]
- 地理设定：[主要地点]
- 特殊设定：[是否有特殊规则、秘境、传承等]

## 前文剧情摘要
{前文摘要}
- 主要情节发展
- 主角实力提升情况
- 当前冲突点
- 主要敌人或挑战

## 本章内容要求
1. **字数要求**：4000-5000字（符合番茄小说日更要求）
2. **节奏要求**：
   - 每章至少包含1-2个爽点（实力提升、战斗胜利、获得宝物等）
   - 保持快节奏，避免拖沓
   - 适当增加战斗场面，增强代入感
3. **修炼要求**：
   - 本章可以涉及修炼、突破、战斗等
   - 修炼描写要具体，不能过于抽象
   - 实力提升要有具体体现
4. **情节要求**：
   - 承接前文剧情，推进主线发展
   - 可以设置新的冲突、敌人或挑战
   - 保持人物性格一致性
   - 推进主角成长线
5. **战斗要求**（如有战斗）：
   - 战斗场面要精彩，有层次感
   - 战斗策略要合理
   - 可以设置反转或意外
6. **平台适配**：
   - 符合番茄小说读者偏好（爽文、节奏快、升级流）
   - 对话要生动
   - 每章结尾可以设置悬念或新的挑战

## 输出格式
标题：[章节标题，10-20字，吸引人，如"第X章：突破瓶颈"或"第X章：激战强敌"]

正文：
[章节正文内容，4000-5000字，段落清晰，对话自然]

---
章节信息：
- 字数：[实际字数]
- 主要情节：[1-2句话概括本章核心情节]
- 实力变化：[主角实力变化情况]
- 关键人物：[本章出现的主要人物]
- 重要事件：[本章发生的重要事件]
```

## 修炼体系示例

### 示例1：传统修真体系
```
修炼体系设定：
- 境界划分：淬体境、凝气境、筑基境、金丹境、元婴境、化神境、合体境、大乘境、渡劫境
- 当前境界：筑基境中期
- 修炼方式：吸收天地灵气，运转功法
- 特殊设定：主角拥有特殊体质"混沌体"
```

### 示例2：创新修炼体系
```
修炼体系设定：
- 境界划分：一星武者、二星武者...九星武者、武师、大武师、武王、武皇、武帝
- 当前境界：三星武者
- 修炼方式：炼体、炼气、炼魂
- 特殊设定：主角拥有"吞噬天赋"，可以吞噬他人修为
```

假设要生成《逆天武神》第25章：

```
你是一位专业的网络小说作家，擅长创作玄幻修真小说，作品在番茄小说平台广受欢迎。

## 任务目标
为玄幻修真小说《逆天武神》创作第25章内容。

## 原创性要求（必须严格遵守）
[使用上面的原创性要求]

## 小说基本信息
- 小说名称：逆天武神
- 主角姓名：叶辰
- 类型：玄幻修真
- 风格：热血升级，装逼打脸

## 修炼体系设定
- 境界划分：淬体境、凝气境、筑基境、金丹境、元婴境、化神境
- 当前境界：筑基境中期
- 修炼方式：吸收天地灵气，运转《九转玄功》
- 特殊设定：主角拥有"混沌体"，修炼速度是常人的十倍

## 人物设定
- 主角：叶辰，性格坚毅，有仇必报，从废材逆袭
- 当前实力：筑基境中期，掌握多种武技
- 主要功法：《九转玄功》《破天拳》《疾风步》
- 主要配角：师父云老、师妹林雪、敌人赵天（筑基境后期）

## 世界观设定
- 世界背景：天武大陆，以武为尊
- 主要势力：天云宗（主角所在）、血刀门（敌对）
- 地理设定：天云山脉、天云城

## 前文剧情摘要
前24章讲述了叶辰从废材逆袭，获得奇遇，实力不断提升。第24章，叶辰在宗门大比中击败强敌，获得筑基丹，准备突破到筑基境后期。但此时血刀门来袭，宗门陷入危机。

## 本章内容要求
[使用上面的内容要求]

## 输出格式
[使用上面的输出格式]
```

## 注意事项

1. **修炼体系要独特**：避免使用过于常见的境界划分
2. **战斗要精彩**：战斗场面要有层次，不能过于简单
3. **实力提升要合理**：不能过于快速，要有过程
4. **世界观要完整**：设定要前后一致，不能矛盾
5. **人物要立体**：避免脸谱化，给角色更多层次
