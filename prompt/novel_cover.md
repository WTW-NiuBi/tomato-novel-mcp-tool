# 小说封面设计Prompt模版

专门用于AI生成小说封面，针对**番茄小说平台**优化，强调视觉吸引力和点击率。

## 基础Prompt结构

```
封面类型：[封面类型]
小说信息：[小说基本信息]
视觉要求：[视觉设计要求]
风格要求：[艺术风格要求]
技术规格：[尺寸和分辨率要求]
```

## 完整Prompt模版

```
你是一位专业的数字插画师和封面设计师，擅长创作引人注目的小说封面，作品在番茄小说平台广受欢迎。

## 任务目标
为小说《{小说名称}》设计一个引人注目、吸引点击的封面。

## 原创性要求（必须严格遵守）
1. 完全原创设计，不得抄袭、模仿或直接使用任何现有小说封面
2. 不得使用任何知名小说、电影、游戏的角色形象或场景
3. 人物形象、场景设计必须原创
4. 色彩搭配、构图设计要独特
5. 生成后需要确保与现有封面不相似

## 小说基本信息
- 小说名称：{小说名称}
- 小说类型：{小说类型，如：都市言情/玄幻修真/都市系统流等}
- 主要人物：{主要人物描述}
- 故事背景：{故事背景简述}
- 核心元素：{核心元素，如：霸道总裁、系统、修仙等}

## 视觉设计要求（重点：引人注目）

### 1. 主体元素
- **主要人物**：{人物描述，如：英俊的男性/美丽的女性，具体特征}
- **场景设置**：{场景描述，如：现代都市/古代宫殿/玄幻世界}
- **核心道具**：{重要道具，如：系统界面、法宝、契约书等}

### 2. 构图要求
- **构图方式**：中心构图或黄金分割构图，主体突出
- **视觉焦点**：主要人物或核心元素位于视觉中心
- **层次感**：前景、中景、背景层次分明
- **留白处理**：为标题文字预留清晰空间（上方或下方）

### 3. 色彩要求（关键：吸引眼球）
- **主色调**：{主色调，如：暖色调（红/橙/金）或冷色调（蓝/紫/银）}
- **对比度**：高对比度，色彩鲜明，视觉冲击力强
- **色彩搭配**：{具体配色方案，如：红金配、蓝紫配、黑白金配等}
- **渐变效果**：适当使用渐变，增加视觉层次
- **强调色**：使用1-2种强调色突出重点元素

### 4. 氛围营造
- **情感氛围**：{氛围描述，如：浪漫唯美/热血激昂/神秘悬疑/轻松幽默}
- **光影效果**：适当的光影对比，增强立体感
- **动态感**：可以适当表现动态元素（如：飘动的发丝、飞舞的花瓣、能量波动等）

## 艺术风格要求

### 风格选择（根据小说类型）
- **都市言情类**：现代插画风格、唯美浪漫、细节丰富、数字绘画
- **玄幻修真类**：国风插画、仙侠风格、水墨渲染、气势磅礴
- **都市系统流**：现代科技感、赛博朋克元素、未来感、炫酷风格
- **悬疑推理类**：暗黑风格、神秘氛围、光影对比强烈、悬疑感

### 细节要求
- **精细度**：高精细度，细节丰富
- **质感**：材质质感真实（如：丝绸、金属、皮肤等）
- **人物表情**：表情生动，符合人物性格
- **背景细节**：背景丰富但不喧宾夺主

## 技术规格要求

### 尺寸规格
- **标准尺寸**：750x1000像素（3:4比例，适合手机阅读）
- **分辨率**：至少300 DPI，确保清晰度
- **文件格式**：PNG或JPG，支持透明背景（如需要）

### 文字区域预留
- **标题区域**：上方或下方预留清晰区域，用于放置小说标题
- **文字可读性**：确保背景不会影响文字清晰度
- **标题样式建议**：{标题样式，如：粗体、描边、阴影等}

## 平台适配要求（番茄小说）

1. **视觉冲击力**：封面必须第一时间吸引眼球，让人有点击欲望
2. **缩略图效果**：在小尺寸缩略图下也要清晰可见
3. **类型识别**：一眼就能看出小说类型（言情/玄幻/系统流等）
4. **情感共鸣**：封面要能传达小说的核心情感
5. **差异化**：与同类小说封面有明显区别，避免同质化

## 输出要求

生成一个完整的小说封面设计，包含：
1. 主体画面（人物+场景）
2. 色彩搭配方案
3. 构图布局说明
4. 风格特点描述

## 设计技巧总结

### 提高点击率的关键要素

1. **高对比度色彩**：使用鲜明对比的色彩组合
2. **突出主体**：主要人物或元素要清晰突出
3. **情感共鸣**：封面要能传达小说的核心情感
4. **类型识别**：一眼就能看出小说类型
5. **差异化设计**：与同类小说有明显区别
6. **缩略图优化**：在小尺寸下也要清晰可见

### 不同小说类型的封面特点

| 类型 | 主色调 | 核心元素 | 风格特点 |
|------|--------|----------|----------|
| 都市言情 | 暖色调（金/粉/红） | 男女主角、浪漫场景 | 唯美浪漫、奢华感 |
| 都市系统流 | 冷色调+金色（蓝/紫/金） | 系统界面、科技元素 | 科技感、未来感、炫酷 |
| 玄幻修真 | 金色+深色（金/蓝/紫） | 武者、武器、能量 | 国风、仙侠、气势磅礴 |
| 悬疑推理 | 暗色调（黑/灰/深蓝） | 神秘元素、光影对比 | 暗黑、神秘、悬疑感 |

## 注意事项

1. **原创性**：确保设计完全原创，避免与现有封面相似
2. **文字可读性**：预留清晰的标题区域，确保文字清晰可读
3. **平台规范**：遵守番茄小说平台的封面规范
4. **多次迭代**：初次生成可能不完美，需要多次调整优化
5. **A/B测试**：可以生成多个版本，测试哪个效果更好

---

**使用建议**：
1. 根据小说类型选择对应的示例模板
2. 填充具体的小说信息
3. 根据实际情况调整色彩、风格等要求
4. 生成后检查是否符合要求，必要时调整Prompt重新生成
