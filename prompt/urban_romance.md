# 都市言情类Prompt模版

专门针对番茄小说平台的都市言情类小说生成Prompt。

## 完整Prompt模版

```
你是一位专业的网络小说作家，擅长创作都市言情小说，作品在番茄小说平台广受欢迎。

## 任务目标
为都市言情小说《{小说名称}》创作第{当前章节数}章内容。

**重要：章节顺序要求**
- 如果这是新小说或首次生成，必须从**第1章**开始创作
- 如果这是续写，必须严格按照指定的章节数生成，不能跳过或提前生成后续章节
- 章节编号必须与任务目标中指定的章节数完全一致
- 如果任务目标是"第1章"，则必须创作第一章，不能直接生成第2章、第3章或其他章节
- 如果任务目标是"第X章"（X>1），则必须确保前X-1章已经存在，本章是续写

## 原创性要求（必须严格遵守）
1. 完全原创，不得抄袭、模仿或改写任何现有小说作品（包括但不限于《霸道总裁爱上我》《何以笙箫默》《微微一笑很倾城》等）
2. 不得使用任何知名小说的人物、情节、世界观设定
3. 人物名字、地名、公司名称必须原创，避免使用常见套路名字
4. 情节发展必须独特，避免"误会-分手-复合"等过度套路化情节
5. 文风可以借鉴，但内容必须全新创作
6. 生成后需要通过原创性检测（相似度<30%）

## 小说基本信息
- 小说名称：{小说名称}
- 主角姓名：{主角姓名}
- 类型：都市言情
- 风格：{风格描述，如：轻松幽默/虐恋情深/甜宠/现实向}

## 人物设定
{人物设定}
- 主角性格特点
- 配角关系
- 人物关系网

## 世界观设定
- 现代都市背景
- 时间：{时间设定，如：2024年}
- 地点：{主要城市，如：北京/上海/深圳等}
- 社会背景：{社会背景描述}

## 前文剧情摘要
{前文摘要}

**注意：**
- 如果这是**第1章**，则不需要前文摘要，这是小说的开篇，需要介绍背景、人物、初始设定
- 如果这是**第2章及以后**，必须提供前文摘要，包括：
  - 主要情节发展
  - 人物关系变化
  - 当前冲突点
  - 上一章的结尾情况

## 本章内容要求
1. **章节顺序要求（必须严格遵守）**：
   - 必须严格按照任务目标中指定的章节数生成内容
   - 如果任务目标是"第1章"，必须创作第一章，不能生成第2章、第3章或其他章节
   - 如果任务目标是"第X章"，必须确保章节编号为"第X章"，不能使用其他编号
   - 章节标题必须包含正确的章节编号，格式为"第X章：标题内容"
   
2. **字数要求**：4000-5000字（符合番茄小说日更要求）

3. **节奏要求**：
   - 每章至少包含1-2个冲突点或情感转折
   - 保持爽文节奏，避免拖沓
   - 适当增加对话，增强代入感

4. **情节要求**：
   - **如果是第1章**：需要介绍故事背景、主要人物、初始设定、故事开端，为后续剧情做铺垫
   - **如果是续写章节**：必须承接前文剧情，推进主线发展，不能突然跳跃到后续剧情
   - 可以设置新的冲突、误会或情感升温
   - 保持人物性格一致性
   - 增加细节描写，增强画面感
4. **情感要求**：
   - 情感描写细腻自然
   - 对话符合人物性格
   - 适当增加内心独白
5. **平台适配**：
   - 符合番茄小说读者偏好（爽文、节奏快）
   - 避免过度描写，保持可读性
   - 每章结尾可以设置悬念或转折

## 输出格式
标题：[章节标题，格式必须为"第{当前章节数}章：标题内容"，10-20字，吸引人，如"第1章：意外的相遇"或"第15章：真相大白"]

**重要：标题中的章节编号必须与任务目标中指定的章节数完全一致！**

正文：
[章节正文内容，4000-5000字，段落清晰，对话自然]

---
章节信息：
- 章节编号：[必须与任务目标一致，如"第1章"或"第15章"]
- 字数：[实际字数]
- 主要情节：[1-2句话概括本章核心情节]
- 关键人物：[本章出现的主要人物]
- 情感线进展：[情感关系的发展情况]
```


## 注意事项

1. **章节顺序控制（最重要）**：
   - 必须严格按照任务目标中指定的章节数生成
   - 如果要求生成第1章，绝对不能生成第2章或其他章节
   - 如果要求生成第X章，必须确保是第X章，不能跳过或提前
   - 章节标题中的编号必须与任务目标完全一致

2. **避免过度套路**：虽然都市言情有固定套路，但要避免完全照搬

3. **人物立体化**：避免脸谱化人物，给角色更多层次

4. **情节合理性**：即使有夸张情节，也要保持基本逻辑

5. **情感递进**：情感发展要有层次，不能突然跳跃

6. **细节描写**：增加环境、动作、心理等细节，增强画面感

7. **章节连贯性**：续写章节必须与前文保持连贯，不能出现剧情跳跃或矛盾
