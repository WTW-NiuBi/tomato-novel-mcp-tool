# Prompt 模版库

本目录存放所有用于AI生成小说的Prompt模版，专门针对**番茄小说平台**优化。

## 目录结构

### 内容生成Prompt
- `base_prompt.md` - 基础Prompt模版和通用规则
- `urban_romance.md` - 都市言情类Prompt
- `urban_system.md` - 都市系统流Prompt（番茄热门）
- `fantasy_cultivation.md` - 玄幻修真类Prompt
- `suspense_thriller.md` - 悬疑推理类Prompt

### 封面设计Prompt
- `novel_cover.md` - 小说封面设计Prompt（AI生成封面）

## 使用说明

### 内容生成
1. **选择垂类**：根据目标小说类型选择对应的Prompt文件
2. **填充变量**：替换Prompt中的占位符（如{前文摘要}、{人物设定}等）
3. **调用AI**：使用选定的Prompt调用AI模型生成内容
4. **原创性检测**：生成后必须进行原创性检测（相似度<30%）

### 封面设计
1. **选择类型**：根据小说类型选择对应的封面设计示例
2. **填充信息**：替换小说名称、人物、背景等信息
3. **调用AI绘图**：使用Prompt调用AI绘图工具（如Midjourney、DALL-E、Stable Diffusion等）
4. **优化迭代**：根据生成结果调整Prompt，多次优化直到满意

## 番茄小说平台特点

- **免费阅读模式**：靠广告分成，需要高阅读量
- **更新要求**：日更4000字以上
- **热门类型**：都市言情、都市系统流、玄幻修真
- **读者偏好**：爽文、节奏快、冲突多、代入感强

## Prompt设计原则

1. **强调原创性**：每个Prompt都包含原创性要求
2. **符合平台调性**：针对番茄小说的读者偏好优化
3. **保持连贯性**：支持前文摘要，保持剧情连贯
4. **控制字数**：确保生成4000-5000字章节
5. **节奏把控**：要求每章有冲突、有爽点
