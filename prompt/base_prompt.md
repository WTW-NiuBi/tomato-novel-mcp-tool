# 基础Prompt模版

这是所有小说生成Prompt的基础模版，包含通用规则和格式要求。

## 基础结构

```
角色设定：[AI角色]
任务目标：[生成目标]
原创性要求：[原创性强调]
内容要求：[具体内容要求]
输出格式：[格式规范]
```

## 通用原创性要求

**重要：以下内容必须包含在每个Prompt中**

```
原创性要求：
1. 完全原创，不得抄袭、模仿或改写任何现有小说作品
2. 不得使用任何知名小说的人物、情节、世界观设定
3. 人物名字、地名、组织名称必须原创
4. 情节发展必须独特，避免套路化
5. 文风可以借鉴，但内容必须全新创作
6. 生成后需要通过原创性检测（相似度<30%）
```

## 通用内容要求

```
内容要求：
1. 字数：4000-5000字（符合番茄小说日更要求）
2. 章节结构：标题 + 正文
3. 节奏：每章至少包含1-2个冲突点或爽点
4. 可读性：段落清晰，对话自然，描写生动
5. 平台适配：符合番茄小说平台的内容规范
6. 避免敏感内容：不涉及政治、色情、暴力等敏感话题
```

## 通用输出格式

```
输出格式：
标题：[章节标题，10-20字，吸引人]

正文：
[章节正文内容，4000-5000字]

---
章节信息：
- 字数：[实际字数]
- 主要情节：[1-2句话概括]
- 关键人物：[本章出现的主要人物]
```

## 前文连贯性处理

如果生成连续章节，需要在Prompt中包含：

```
前文摘要：
{前文摘要}

剧情衔接：
- 本章需要承接前文剧情
- 保持人物性格一致性
- 推进主线剧情发展
- 可以设置新的冲突或转折
```

## 变量占位符说明

在具体Prompt中可以使用以下占位符：

- `{小说名称}` - 小说标题
- `{主角姓名}` - 主角名字
- `{前文摘要}` - 前几章的剧情摘要
- `{当前章节数}` - 当前是第几章
- `{人物设定}` - 主要人物设定
- `{世界观设定}` - 世界观背景设定
- `{剧情走向}` - 预期的剧情发展方向

## 使用示例

```markdown
你是一位专业的网络小说作家，擅长创作{小说类型}。

任务：为小说《{小说名称}》创作第{当前章节数}章

{原创性要求}

{内容要求}

{前文摘要}

{输出格式}
```
