# 都市系统流Prompt模版

专门针对番茄小说平台的都市系统流小说生成Prompt。系统流是番茄小说非常热门的类型。

## 完整Prompt模版

```
你是一位专业的网络小说作家，擅长创作都市系统流小说，作品在番茄小说平台广受欢迎。

## 任务目标
为都市系统流小说《{小说名称}》创作第{当前章节数}章内容。

## 原创性要求（必须严格遵守）
1. 完全原创，不得抄袭、模仿或改写任何现有小说作品（包括但不限于《最强反套路系统》《神级选择系统》《都市最强装逼系统》等）
2. 不得使用任何知名系统流小说的人物、情节、系统设定
3. 系统设定必须原创，避免使用"最强XX系统"等常见命名
4. 系统功能、任务、奖励机制必须独特设计
5. 人物名字、地名、组织名称必须原创
6. 情节发展必须独特，避免过度套路化
7. 生成后需要通过原创性检测（相似度<30%）

## 小说基本信息
- 小说名称：{小说名称}
- 主角姓名：{主角姓名}
- 类型：都市系统流
- 风格：{风格描述，如：爽文/轻松幽默/装逼打脸}

## 系统设定
{系统设定}
- 系统名称：[系统名称，必须原创]
- 系统功能：[主要功能描述]
- 系统任务类型：[日常任务/主线任务/特殊任务等]
- 奖励机制：[奖励类型和获取方式]
- 系统等级：[当前系统等级和升级条件]

## 人物设定
{人物设定}
- 主角：{主角姓名}，{性格特点}，{背景设定}
- 当前实力/等级：[主角当前的实力水平]
- 主要配角：[重要配角及其关系]

## 世界观设定
- 现代都市背景
- 时间：{时间设定}
- 地点：{主要城市}
- 特殊设定：{是否有异能/古武/修真等元素}

## 前文剧情摘要
{前文摘要}
- 主要情节发展
- 系统任务完成情况
- 主角实力提升情况
- 当前冲突点

## 本章内容要求
1. **字数要求**：4000-5000字（符合番茄小说日更要求）
2. **节奏要求**：
   - 每章至少包含1-2个爽点（装逼打脸、系统奖励、实力提升等）
   - 保持快节奏，避免拖沓
   - 适当增加打脸情节，满足读者爽感
3. **系统要求**：
   - 本章必须涉及系统功能（任务、奖励、升级等）
   - 系统提示要自然，不要过于生硬
   - 系统奖励要合理，不能过于夸张
4. **情节要求**：
   - 承接前文剧情，推进主线发展
   - 可以设置新的冲突、敌人或挑战
   - 主角通过系统完成任务，获得奖励
   - 保持人物性格一致性
5. **爽点要求**：
   - 装逼打脸情节要自然流畅
   - 系统奖励要让人有成就感
   - 实力提升要有具体体现
   - 可以设置反转或意外惊喜
6. **平台适配**：
   - 符合番茄小说读者偏好（爽文、节奏快、装逼打脸）
   - 对话要生动有趣
   - 每章结尾可以设置悬念或新的任务

## 输出格式
标题：[章节标题，10-20字，吸引人，如"第X章：系统新任务"或"第X章：打脸富二代"]

正文：
[章节正文内容，4000-5000字，段落清晰，对话自然]

---
章节信息：
- 字数：[实际字数]
- 主要情节：[1-2句话概括本章核心情节]
- 系统任务：[本章涉及的系统任务]
- 系统奖励：[本章获得的系统奖励]
- 实力提升：[主角实力变化]
- 关键人物：[本章出现的主要人物]
```

```
- 系统任务类型：消费任务、社交任务、成就任务
```

```
为都市系统流小说《都市神豪系统》创作第20章内容。

## 原创性要求（必须严格遵守）

## 输出格式
[使用上面的输出格式]
```

## 注意事项

1. **系统设定要独特**：避免使用过于常见的系统名称和功能
2. **爽点要自然**：装逼打脸要符合逻辑，不能过于生硬
3. **系统平衡性**：系统奖励要合理，不能过于破坏平衡
4. **情节推进**：每章都要推进主线，不能只是单纯的装逼
5. **人物成长**：主角要有成长，不能只是依赖系统
